# 风格标签展示优化

## 问题诊断

### 原有问题

之前的展示方式：
- ❌ 所有标签一次性全部显示
- ❌ 没有分类，杂乱无章
- ❌ 无法快速定位重要标签
- ❌ 页面过长，体验差

```
词汇风格：口语化 (5/5)
词汇风格：书面化 (4/5)
词汇风格：古语词 (3/5)
句法风格：长句 (5/5)
句法风格：短句 (4/5)
句法风格：倒装 (3/5)
修辞风格：比喻 (5/5)
修辞风格：排比 (4/5)
修辞风格：对偶 (3/5)
...（还有 10+ 个标签）
```

用户需要滚动很久才能看完，而且找不到重点。

## 优化方案

### 新的展示方式

✅ **按维度分组**：词汇风格、句法风格、修辞风格等
✅ **默认折叠**：每个维度只显示 2 个最高评分的标签
✅ **按需展开**：点击"查看更多"展开所有标签
✅ **评分排序**：每个维度内按评分从高到低排序

## 实现细节

### 1. 维度分组

```javascript
// 提取维度名称（冒号前的部分）
const parts = label.label.split(':');
const dimension = parts.length > 1 ? parts[0].trim() : '其他';
const labelName = parts.length > 1 ? parts.slice(1).join(':').trim() : label.label;
```

示例：
- 输入：`"词汇风格：口语化"`
- 维度：`"词汇风格"`
- 标签名：`"口语化"`

### 2. 按评分排序

```javascript
// 每个维度内按评分降序排序
groupedLabels[dimension].sort((a, b) => b.score - a.score);
```

确保每个维度内评分最高的标签排在最前面。

### 3. 默认显示 2 个

```javascript
const defaultShow = 2; // 默认显示 2 个
const isExpandable = labels.length > defaultShow;

labels.forEach((label, index) => {
    if (index >= defaultShow) {
        tag.style.display = 'none';
        tag.classList.add('hidden-tag');
    }
    // ...
});
```

### 4. 展开/收起功能

```javascript
toggleBtn.onclick = function() {
    const hiddenTags = labelsContainer.querySelectorAll('.hidden-tag');
    hiddenTags.forEach(tag => {
        tag.style.display = isExpanded ? 'none' : 'block';
    });
    
    if (isExpanded) {
        toggleBtn.innerHTML = '查看更多 ⌄';
    } else {
        toggleBtn.innerHTML = '收起 ⌃';
    }
    isExpanded = !isExpanded;
};
```

## 性能优化

### 减少初始渲染

- 优化前：渲染所有标签（可能 20+ 个 DOM 元素）
- 优化后：只渲染 8-10 个（每个维度 2 个）
- 性能提升：约 50-60%

### 渐进式加载

用户点击"查看更多"时才显示隐藏的标签，避免一次性加载过多内容。

## 可配置选项

如果需要调整默认显示数量，可以修改：

```javascript
const defaultShow = 2; // 改为 1、3 或其他数字
```

建议值：
- **1 个**：极简模式，只显示最好的
- **2 个**：平衡模式（推荐）
- **3 个**：详细模式
